Файл product_page.php возвращает json с информацией, подогнанной под details.php (карточка продукта).

На вход берется GET-параметр artikul (в бд заведен товар с артикулом, названием и всем остальным аналогично details.php)

Возвращается массив:
    - наименование (name);
    - описание (description);
    - адрес изображения (img_src) - формата /images/catalog/9.jpg;
    - цена (price);
    - массив с информацией по размерам (size_data):
        - номер размера (name);
        - наличие (is_available): 1 - есть на складе, 0 - нет;
        - количество (amount).
